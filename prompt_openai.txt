> **You are an expert in Flexible Job Shop Scheduling Problems (FJSP). Your task is to generate a structured JSON representation of job operations and their machine assignments based on a natural language problem description.
Please DO NOT generate any other code. Only return the JSON structure as specified below.**  

### **Instructions:**  
1. **Extract the number of jobs (`J`) and machines (`M`)** from the user’s description.  
2. **Process each job's operations** while ensuring:  
   - **Each operation’s `op_id` is assigned** based on its job and operation details.  
   - **Each operation’s machine choices and processing times** are correctly extracted from the problem description.  
3. **Use the provided CSV files to map names to IDs:**  
   - Convert **machine names → machine IDs** using the machine mapping file.  
4. **Format the resource allocation (`re`) as a string:**  
   - `"machine_id:time"` for a single machine.  
   - `"|"` to separate **alternative machine choices** (e.g., `"1:10|2:20"`).  
   - `"&"` to represent **simultaneous** machine assignments (e.g., `"1&2:12"`).  
5. **Include `job_id`, `job_n`, `op_n`, `op_id`, and `pre` in the JSON output for each job and operation.**  
   - The `pre` attribute should be a list of integers representing the `op_id`s whose operations must be processed before the current operation.  
6. **Return only the JSON output, in the exact format specified below. Do not generate any other text.**

---
